# Zahoor Flour Mills 🌾

A full stack e-commerce web application for a local flour mill based in Sahiwal, Punjab, Pakistan.

## Technologies Used
- Frontend: React.js
- Backend: Node.js + Express.js
- Database: MongoDB
- Authentication: JWT + bcryptjs
- Payment: Stripe API

## Features
- User Registration & Login
- Products Browsing & Shopping Cart
- Checkout & Payment (COD + Stripe)
- Customer Reviews System
- Wholesale Inquiry Form
- Admin Dashboard with full CRUD operations
